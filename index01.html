新車交車前檢查表

專門針對義大利原廠電控與底盤架構量身打造之高階點檢表單。

一、核心管理概念說明

為建立嚴謹的車輛品質管控體系，必須嚴格釐清「庫存維護」與「新車交車前檢查`」的本質差別。以實體「新車整備管制卡 (PDI CONTROL)」作為總開關，確保每台車進入正確路徑，杜絕管理漏洞。

📋 REFERENCE IMAGE: PDI ARRIVAL ARRANGEMENT (A02)

管制卡屬性分流基準

概念一【庫存車】：
新車到貨超過 14 天未綁定訂單者，強制導入後端庫存維護管理程序。
概念二【待交車】：
已綁定訂單並準備交付客戶者，即刻切換至交車前黃金 48 小時檢查流程。
📋 REFERENCE IMAGE: SA IDENTIFICATION DIAGRAM (A03)

【庫存車】巡檢空間動線與防呆機制

1. 內湖廠區三大特定巡檢區域：
  • 一樓側邊
  • 頂樓停放區
  • 樓上庫存區（技師必須特別注意）
2. 雙軌表單防呆管理位置：
  • 位置一：放置於車輛內的儀表板顯眼處（隨車查核）。
  • 位置二：集中於值班櫃檯進行行政進度掌控與稽核。
📋 REFERENCE IMAGE: STOCK MAINTENANCE MANAGEMENT (A01)

庫存技術核心：電瓶管理數據留痕

依照維護單週期落實檢查，重點在於 12V 與 48V 的電瓶檢測與報告列印。每一次電瓶量測之檢測報告，皆必須即時上傳至 ARGOS 伺服器。

📋 REFERENCE IMAGE: 12V & 48V BATTERY LOG (A04)

總代理交車文件上傳清單 (依車系與車型分流)

【車系分類一：Peugeot / Citroën (PSA)】
  • 常規車型（必須備妥標準 5 張單上傳 DMS）：
    • 發貨驗收單
    • 12V 電池檢測報告（48 小時內）
    • 48V 電池檢測報告（48 小時內）
    • 新車交車前檢查表
    • 經銷商庫存維護單（隨車之歷次維護紀錄）
  • 特定車型（408 / 3008 / 5008）之加強規範：
    除上述 5 張單外，必須額外加附第 6 張表格：「新車交車前功能檢查項目表」。
【車系分類二：Alfa Romeo】指標車型（Giulia / Stelvio 專屬規範）：

由於電控與品牌架構特殊，不適用常規交車檢查表。必須依總代理規範，將第四項文件替換為專屬表單：

  • 發貨驗收單
  • 12V 電池檢測報告（48 小時內）
  • 「Giulia / Stelvio 專屬全新交車前檢查表」
  • 經銷商庫存維護單
📋 REFERENCE IMAGE: PDI FILE STORAGE REGULATION (A05)

⚠️ 經銷商通報防範條款與處罰機制

若未確實依照規定流程作業，致使後續發生以下情事者，其損失將由該經銷商全額自行負責，總代理恕不予補償：

  • 保固申請遭拒：因未留存檢查紀錄或作業程序不符規範，導致原廠判定保固不予通過者。
  • 原廠稽核扣款：於原廠年度稽核或專案查核中, 因檢查清單缺失、造假或漏項，導致之追溯扣款、罰鍰或相關財務損失者。
