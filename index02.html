一、銷售時機規範與時間防呆

⏱️ 銷售時機與資格門檻： 新車交車前、或新車保固（3年/10萬公里）到期日前 90 天內均可購買。
🚨 系統鎖死防呆（0~33 個月限時）： 當車齡進入第 34、35、36 個月時，總代理系統將硬性鎖死，絕對不能受理購買。
💰 銷售人員獎勵機制： 成功銷售延長保固，均可獲得獎勵金 $3,500 元/台（已內含在售價內）。

OPERATIONAL GUIDELINE // REF: B11

01. DMS系統送件申請

啟動延長保固流程的第一步。請第一線同仁登入 DMS 系統「會員中心」，進入「合約權益商品銷售」專區。

此階段必須將車主資訊與購買合約完整建檔，並同步於後台附件欄位上傳「延長保固契約書」、「延長保固車況檢視表」以及「車況檢查照片」共三份核心文件，做為總代理審查的必備依據。

TECHNICAL VERIFICATION // REF: B13

02. 實車車輛檢查

🚨 延保防呆機制的實體根基：

進廠加購延保之車輛必須開入頂高機位置，由技師依照「車況檢查表」項目執行嚴格的實車檢驗。

查驗核心包含：確認過去所有保養紀錄皆在原廠施作、檢視全車電路系統與機件結構是否完整，並嚴格排除任何非原廠改裝配件。查驗完成後必須現場拍攝清晰的實車外觀與重要總成照片作為查核佐證。

CLIENT ENGAGEMENT // REF: B02

03. 向車主說明與合約確認

⚖️ 履行法律責任與信任建立：

這是建立車主信任感並履行法律責任的關鍵溝通節點。服務專員（SA）應當面引導車主閱讀，並詳細解說本次加購的延長保固範疇、權益期限、以及相關不保事項（改裝、外力破壞等排除條款）。

經車主充分理解並確認無誤後，引導車主於「延長保固契約書」正本上親筆簽名具結。契約書正本由廠內妥善留存歸檔，並將清晰掃描件上傳 DMS 系統附件備查。

FINANCIAL SETTLEMENT // REF: B12

04. 結帳與開立發票

銷售流程的財務終點。請會計 or 收銀同仁嚴格按照全車系延長保固管制之標準含稅費率進行結帳，並當場開立合法統一發票。

發票品項必須清晰登載延長保固對應之合約金額，確認向車主收取足額費用後，將發票聯與工單合併歸入檔案，以備每月 25 號清單總表回傳時與總代理進行請款核對。

SUBMISSION OVERVIEW // REF: B15

05. 提供送審文件給總代理

為簡化前線與後台對帳流程，全車系送件規範已全面同步：

  • Alfa Romeo 車系 與 Peugeot / Citroën / Jeep 車系：原則上這兩種品牌在接待收單時，皆必須將三張應備附件（契約書、車況檢視表、車況檢查表照片）直接線上上傳送至 DMS 系統內做為總代理審查之依據。

🚨 審查退件防呆：若上傳之附件有模糊 or 缺件，將全單退回服務廠，需修正後重新送審。

MONTHLY REGULATION // REF: B14

06. 每月25號定期清單回傳

服務廠必須於每月 25 號前，確實填報彙整當月份所有完工結帳之【延長保固車輛購買清單總表】明細 Excel 檔。

核對無誤後，請將該清單總表以 E-mail 方式正式提供給總代理窗口 Jack Wu。總代理售後服務部將以此總表與服務廠進行核對，並於次月份據此向各服務廠進行請款與對帳程序。
